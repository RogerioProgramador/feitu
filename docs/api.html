API Reference

Base URL: /api — Todas as rotas (exceto /auth/**) exigem Authorization: Bearer <token>

📋

Erros são retornados como ProblemDetail (RFC 7807):

{ "type": "about:blank", "title": "Unprocessable Entity", "status": 422, "detail": "mensagem" }

Auth

POST /api/auth/register

Cria uma nova conta.

Body

{
  "email": "usuario@exemplo.com",
  "senha": "minimo8chars"
}

Resposta 201 Created

{ "token": "eyJhbGci..." }

Erros

400422
POST /api/auth/login

Autentica e retorna o token.

Body

{
  "email": "usuario@exemplo.com",
  "senha": "minimo8chars"
}

Resposta 200 OK

{ "token": "eyJhbGci..." }

Erros

401

Workspaces

GET /api/workspaces

Lista os workspaces do usuário, ordenados por ordem.

Resposta 200 OK

[
  { "id": "uuid", "nome": "Trabalho", "cor": "#A7C7E7", "ordem": 1 }
]
POST /api/workspaces

Cria um novo workspace. A ordem é atribuída automaticamente (último+1).

Body

{ "nome": "Trabalho", "cor": "#A7C7E7" }

Resposta 201 Created

Objeto WorkspaceResponse

PUT /api/workspaces/{id}

Atualiza nome e/ou cor de um workspace.

Body

{ "nome": "Trabalho Remoto", "cor": "#B5EAD7" }

Resposta 200 OK

Objeto WorkspaceResponse

Erros

404
PUT /api/workspaces/reorder

Reordena os workspaces. O array ids define a nova ordem (posição 0 = primeiro).

Body

{ "ids": ["uuid-b", "uuid-a", "uuid-c"] }

Resposta 204 No Content

DELETE /api/workspaces/{id}

Remove o workspace e todas as suas tarefas e segmentos de tempo.

Resposta 204 No Content

Erros

404

Tarefas

GET /api/workspaces/{workspaceId}/tarefas

Lista todas as tarefas do workspace (qualquer estado).

Resposta 200 OK

[
  {
    "id": "uuid",
    "nome": "Revisar PRs",
    "estado": "IDLE",
    "tempoTotalSegundos": 0,
    "criadoEm": "2026-06-20T10:00:00",
    "concluidoEm": null
  }
]

tempoTotalSegundos é a soma dos segmentos fechados. Para tarefas RUNNING, o frontend incrementa localmente a cada segundo.

POST /api/workspaces/{workspaceId}/tarefas

Cria uma nova tarefa. Se nome for omitido, o backend usa "Nova tarefa".

Body (opcional)

{ "nome": "Escrever testes" }

Resposta 201 Created

Objeto TarefaResponse

PATCH /api/tarefas/{id}/nome

Renomeia uma tarefa.

Body

{ "nome": "Escrever testes unitários" }

Resposta 200 OK

Objeto TarefaResponse

POST /api/tarefas/{id}/iniciar

Inicia ou retoma o timer. Transições válidas: IDLE → RUNNING, PAUSED → RUNNING.

Resposta 200 OK / Erros

422
POST /api/tarefas/{id}/pausar

Pausa o timer. Transição válida: RUNNING → PAUSED.

Resposta 200 OK / Erros

422
POST /api/tarefas/{id}/parar

Conclui a tarefa. Transições válidas: RUNNING → DONE, PAUSED → DONE. O campo concluidoEm é preenchido.

Resposta 200 OK / Erros

422
POST /api/tarefas/{id}/reativar

Reabre uma tarefa concluída. Transição válida: DONE → IDLE. Limpa concluidoEm.

Resposta 200 OK / Erros

422
DELETE /api/tarefas/{id}

Remove a tarefa e todos os seus segmentos de tempo.

Resposta 204 No Content

Analytics

GET /api/analytics/daily

Retorna o resumo do dia. Query param date=YYYY-MM-DD (opcional, padrão: hoje).

Resposta 200 OK

{
  "data": "2026-06-20",
  "totalSegundos": 14400,
  "porcWorkspace": [
    { "workspaceId": "uuid", "nome": "Trabalho", "cor": "#A7C7E7", "segundos": 9000 },
    { "workspaceId": "uuid", "nome": "Estudos",  "cor": "#B5EAD7", "segundos": 5400 }
  ],
  "timeline": [
    {
      "tarefaId": "uuid",
      "tarefaNome": "Revisar PRs",
      "workspaceCor": "#A7C7E7",
      "inicio": "2026-06-20T09:00:00Z",
      "fim":    "2026-06-20T10:30:00Z"
    }
  ],
  "tarefaMaisLonga": { "id": "uuid", "nome": "Revisar PRs", "segundos": 5400 }
}

porcWorkspace ordenado por tempo desc. tarefaMaisLonga pode ser null.

Push Notifications

POST /api/push/subscribe

Registra uma subscription de push. Idempotente — se já existe para o mesmo endpoint, atualiza.

Body

{
  "endpoint": "https://fcm.googleapis.com/fcm/send/...",
  "keys": { "p256dh": "BNcR...", "auth": "tBHI..." }
}

Resposta 201 Created

DELETE /api/push/unsubscribe

Remove a subscription do endpoint informado.

Body

{ "endpoint": "https://fcm.googleapis.com/fcm/send/..." }

Resposta 204 No Content
